# ❓ Perguntas Frequentes (FAQ)

### 1. Qual a origem da Paper Linux?
A Paper Linux é baseada no **Debian**, herdando sua estabilidade. O ambiente de construção e as ferramentas de interface foram trazidas do **Zorin OS**, resultando em um sistema familiar e poderoso.

### 2. Posso usar este sistema em computadores antigos?
Sim. A Paper Linux foi projetada para ser leve e suportar arquiteturas que exigem eficiência, funcionando bem tanto em 32 quanto em 64 bits dentro de ambientes virtualizados.

### 3. Como instalo novos programas?
Como o sistema é baseado em Debian, você pode usar o terminal:
`sudo apt update && sudo apt install nome-do-pacote`
Ou instalar arquivos `.deb` baixados da internet.

### 4. Por que o arquivo de download é grande (11GB)?
Diferente de uma ISO comum, este arquivo é uma Máquina Virtual **pronta para uso**. Ele já contém o sistema instalado, atualizado e com todas as ferramentas da Paper Linux configuradas, economizando seu tempo de instalação.
